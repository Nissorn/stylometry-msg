## Thai-Stylometry V2: Secure Chat คืออะไร?

**Thai-Stylometry V2** คือระบบแชทความปลอดภัยสูงที่พัฒนาขึ้นสำหรับสภาพแวดล้อมที่ต้องการการพิสูจน์ตัวตนอย่างต่อเนื่อง (Continuous Authentication) ระบบไม่เพียงแต่ตรวจสอบผู้ใช้ตอนเข้าสู่ระบบเท่านั้น แต่ยังคอยวิเคราะห์พฤติกรรมการพิมพ์ข้อความ (Keystroke Dynamics & Writing Style) ตลอดการใช้งาน เพื่อตรวจจับกรณีที่บัญชีถูกยึดครองโดยบุคคลอื่น (Account Takeover) แม้ว่าผู้บุกรุกจะผ่านการล็อกอินมาแล้วก็ตาม

<Aside type="note" title="แนวคิดหลัก">
ระบบตั้งอยู่บนสมมติฐานที่ว่า "ทุกคนมีสไตล์การพิมพ์ที่เป็นเอกลักษณ์เฉพาะตัว" — การลากเสียง การใช้คำอุทาน ความเร็วในการพิมพ์ และโครงสร้างประโยค ล้วนเป็นลายนิ้วมือดิจิทัลที่ยากต่อการปลอมแปลง
</Aside>

## สถาปัตยกรรมระบบ (System Architecture)

ระบบประกอบด้วยสองส่วนหลักที่ทำงานร่วมกัน:

| ชั้น (Layer) | เทคโนโลยี | หน้าที่ |
|---|---|---|
| **Frontend** | Astro + React + TypeScript | UI/UX, WebSocket Client, State Management (Zustand) |
| **Backend API** | FastAPI (Python) | REST Endpoints, JWT Auth, Business Logic |
| **AI Engine** | Python (Mock → Typhoon LLM) | Stylometry Analysis, Calibration Bot |
| **Database** | SQLite (aiosqlite) | User Data, Messages, Audit Logs, OTP Secrets |
| **Security** | pyotp + python-jose | TOTP Generation/Verification, JWT Signing |

```
[User Browser]
     │
     ├── HTTPS REST ──► [FastAPI Backend]
     │                        │
     └── WSS WebSocket ──►    ├── [SQLite Database]
                              ├── [AI Stylometry Engine]
                              └── [Typhoon LLM / Calibration Bot]
```

<Aside type="caution" title="สภาพแวดล้อมการพัฒนา">
ในเวอร์ชันปัจจุบัน โมเดล AI Stylometry ยังอยู่ในโหมด Mock สำหรับการพัฒนา การ Deploy ระบบจริงต้องเชื่อมต่อกับ Typhoon LLM API และ Training Pipeline ของโมเดล
</Aside>

## ผู้พัฒนาและขอบเขตโครงการ

โครงการนี้พัฒนาขึ้นเพื่อการศึกษา ในหัวข้อ **"การประยุกต์ใช้ Stylometry สำหรับ Continuous Authentication ในระบบสื่อสาร"** โดยมีเป้าหมายเพื่อพิสูจน์แนวคิดว่าการวิเคราะห์รูปแบบการเขียนภาษาไทยสามารถนำมาใช้เป็นปัจจัยการพิสูจน์ตัวตนแฝงได้อย่างมีประสิทธิภาพ
