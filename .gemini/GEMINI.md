# Rules for Thai-Stylometry V2
- Framework: FastAPI (Backend) + Astro with React (Frontend)
- Style: Telegram Desktop UI with Tailwind CSS
- Security: Password Hashing (Bcrypt), Message Encryption (Fernet), SQLite.
- AI Logic: 95% Barrier Protocol (5-message rolling window for OWNER only).
- Communication: REST API for Auth/Contacts, WebSockets for Real-time Chat.
